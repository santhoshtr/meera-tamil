Meera Tamil Font
================

A free licensed unicode font for Tamil.


[Download ](https://github.com/santhoshtr/meera-tamil/releases/) latest version.

Also available in Debian and Ubuntu.
```bash
sudo apt-get install fonts-meera-taml
```
